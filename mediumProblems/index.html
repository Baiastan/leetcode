O(1): Constant time - The algorithm runs in constant time regardless of input size.

O(log n): Logarithmic time - The classic example is binary search.

O(n): Linear time - The runtime grows linearly with the size of the input.

O(n log n): Linearithmic time - Seen in algorithms like merge sort or quicksort.

O(n2): Quadratic time - Seen in some algorithms with nested loops.

O(n3): Cubic time - Seen in algorithms with three nested loops, and so on for higher powers.

O(2n): Exponential time - For algorithms like recursive calculations of the Fibonacci numbers.

O(n!): Factorial time - For algorithms like the naive traveling salesman problem.

String Methds

  • charAt(): Returns the character at the specified index.
  • concat(): Concatenates two or more strings and returns a new string.
  • includes(): Determines whether a string contains a certain character or substring.
  • indexOf(): Returns the index of the first occurrence of the specified value, or -1 if not found.
  • lastIndexOf(): Returns the index of the last occurrence of the specified value, or -1 if not found.
  • match(): Used to match a regular expression against a string.
  • repeat(): Returns a new string consisting of the caller repeated the specified number of times.
  • replace(): Replaces matches of a substring or pattern with a replacement.
  • search(): Executes a search for a match between a regular expression and a string.
  • slice(): Extracts a section of a string and returns it as a new string.
  • split(): Splits a string into an array of substrings.
  • startsWith(): Determines whether a string begins with the characters of another string.
  • substring(): Returns the characters in a string between two indices.
  • toLowerCase(): Converts the entire string to lower case.
  • toUpperCase(): Converts the entire string to upper case.
  • trim(): Trims whitespace from the beginning and end of a string.
  • trimStart()/trimLeft(): Trims whitespace from the start of a string.
  • trimEnd()/trimRight(): Trims whitespace from the end of a string.
  • padStart(): Pads the current string from the start with another string until the resulting string reaches the given length.
  • padEnd(): Pads the current string from the end with another string until the resulting string reaches the given length.
